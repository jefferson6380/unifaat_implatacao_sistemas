# TF10 - Migração para Amazon RDS

## Visão Geral

Migração de banco PostgreSQL local Docker para Amazon RDS.

## Arquitetura

### Antes
Docker PostgreSQL local

### Depois
Amazon RDS PostgreSQL

## Como Executar

1. docker compose up -d
2. criar banco
3. executar migração
4. validar

## Resultados

Migração concluída com sucesso.

## Custos

Uso Free Tier AWS.
