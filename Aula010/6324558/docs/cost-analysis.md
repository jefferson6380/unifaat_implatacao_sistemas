# Cost Analysis

## Custo Local (mensal estimado)

| Item | Custo |
|------|-------|
| Hardware (depreciação) | $15,00 |
| Energia elétrica | $8,00 |
| Manutenção e atualizações | $10,00 |
| **Total** | **$33,00** |

## Custo RDS (mensal real)

| Item | Custo |
|------|-------|
| Instância db.t3.micro | $0 (Free Tier) |
| Storage 20GB gp3 | $0 (Free Tier) |
| Backup automático 1 dia | $0 |
| **Total** | **$0** |

> Free Tier válido por 12 meses. Após esse período, custo estimado de $15-20/mês.

## TCO e Recomendações

O RDS elimina custos operacionais de manutenção, atualizações e monitoramento manual. Em ambiente de produção, o custo do RDS é compensado pela redução de horas de administração do banco.
