# Troubleshooting

# Problema 1 - Timeout na conexão

## Erro

Connection timed out

## Solução

Liberação da porta 5432 no Security Group.

---

# Problema 2 - AWS CLI não reconhecido

## Erro

aws: command not found

## Solução

Instalação do AWS CLI e ajuste do PATH.

---

# Problema 3 - pgbench não encontrado

## Erro

pgbench: command not found

## Solução

Instalação do PostgreSQL Client Tools.

---

# Problema 4 - Banco northwind não encontrado

## Solução

Conexão utilizando porta correta do Docker.
