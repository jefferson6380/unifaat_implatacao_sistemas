Calculadora com as quatro operações básicas (soma, subtração, multiplicação e divisão) desenvolvida em Python. Pratica entrada de dados, conversão de tipos primitivos, concatenação de strings e estruturas condicionais.

Jogo da Sorte

Jogo em Python onde o computador sorteia um número aleatório entre 0 e 5 e desafia o usuário a adivinhar. Valida se o número está no intervalo permitido e informa se o jogador venceu ou perdeu. Usa o módulo random.

  • Python
  • Módulo random
  • Lógica de Programação
  • Validação de Entrada

Par ou Ímpar em Python

Jogo onde o usuário escolhe um número e decide se o resultado será par ou ímpar, enquanto o computador gera um número aleatório. O programa soma os dois valores e o jogador continua enquanto acertar. Pratica while e random.

  • Python
  • Laços de Repetição
  • Módulo random
  • Pensamento Computacional
